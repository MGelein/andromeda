Let's Get Started!

As you can see no text is loaded right now. There are two possible solutions to this horrible problem:

1. Load A Text From Perseus

Using the inputfield below you can query the perseus text database to load a text.
You can enter either the official Perseus Name (Catul. 1) or you can enter the URL pointing to the text page. The translation will be automatically loaded for that specific fragment.


2. Load A Saved Text Translation Environment File

If you've worked with this website before and you've remembered to save your work you can restore a previous session by loading the generated Text Translation Environment (.tte) file.
Use the button below to choose a file from your disk to load.

Warning: Saving to and restoring from file is still an experimental feature and should be used with caution!
